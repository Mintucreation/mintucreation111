# Dealer Customer Record — v8 Setup Guide

## What's new in this version
- **Details page now has 3 view modes**, switchable with buttons at the
  top of the page:
  - **☰ List** — one record at a time, every field on its own line
    (the layout from the last update).
  - **▦ Grid** — records shown as compact cards, several per row, good
    for scanning many records at a glance.
  - **▤ Table** — a full spreadsheet-style table with every column,
    same style as the main dashboard table, scrollable sideways on phones.
  Your year/month/date filter works the same across all three views.

## Everything else
No database changes in this update. Deployment, login, the shared
database, reminders, invoice number search, and Excel import/export all
continue to work exactly as before.
